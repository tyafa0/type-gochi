# type-gochi
type-gochi は JavaScript で制作された、キャラクター名の虫食いタイピングゲームです。
与えられた不完全な文字列からキャラクターのフルネームを推測し、正確にタイプする力が求められます。
# 特徴

- 虫食いのお題: 難易度に応じて文字の一部が _ で隠されます。表示されている文字と文脈を頼りに、隠された部分を推測しながら入力してください。
- 3段階の難易度: Easy（空欄率20%）・Normal（50%）・Hard（80%）から選択できます。
- コンボシステム: 連続正解でスコアが倍増し、3・5・10連続クリアで制限時間が延長されます。
- ハイスコア保存: 難易度ごとのハイスコアがローカルに保存されます。

# 遊び方

1. 難易度を選択し、Space キーでゲームを開始します。
2. 画面に表示されたローマ字列（一部が _ で隠されています）をキーボードで入力してください。
3. 制限時間45秒以内にできるだけ多くのキャラクター名を入力しましょう。
